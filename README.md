# 2026-CT-RUN-07

🧪 게임 스토리

RUN:07은 연구실에서 진행된 실험의 일곱 번째 실험체가 된 주인공의 탈출기를 다룬 방탈출 게임이다.

플레이어는 이유도 모른 채 정체불명의 연구실에 갇힌 상태로 눈을 뜨며, 이곳에서 벗어나기 위해 하나씩 단서를 찾아 나서게 된다.

연구실 안에는 총 세 개의 방이 있고, 각 방마다 다른 미션과 퍼즐이 준비되어 있다.

주변을 관찰하고 단서를 조합해 열쇠를 얻으면, 다음 방으로 이동할 수 있다.

단순히 이동만 하는 것이 아니라, 플레이어의 선택과 판단이 탈출의 열쇠가 된다.

모든 미션을 해결하면 마침내 마지막 문 앞에 서게 된다.

RUN:07은 제한된 공간 속에서 점점 자유에 가까워지는 과정을 통해, 긴장감과 몰입감을 함께 느낄 수 있도록 구성된 게임이다.

----------------------------------------------------------------------------------------------------------------------

🎮 게임 방법

🖱 기본 조작

-마우스 클릭: 오브젝트 조사 / 버튼 선택

-키보드 방향키: 맵 화면에서 캐릭터 이동

-Enter: 컴퓨터 퍼즐 비밀번호 입력 확정

----------------------------------------------------------------------------------------------------------------------

🧩 주요 퍼즐 요소

-금고 퍼즐: 숫자 조합을 맞춰 열쇠 획득

-컴퓨터 해킹: 키워드 입력 퍼즐

-전선 퍼즐: 올바른 순서로 전선 자르기

-맵 이동: 방과 방 사이를 이동하며 진행

----------------------------------------------------------------------------------------------------------------------

실행 방법
1. 실행 전 준비

-Python 3.x 설치

-Pygame 라이브러리 설치

----------------------------------------------------------------------------------------------------------------------

참고 사항 (중요)

본 프로젝트는 지정된 한글 글꼴(HCRBatang-Bold.ttf)을 사용합니다.

게임 실행 전 해당 폰트 파일을 프로젝트 폴더(main.py와 동일한 경로)에 추가해야 합니다.

(해당 폰트 파일(HCRBatang-Bold.ttf)은 GitHub 용량 제한으로 인해

저장소에 포함되어 있지 않으며, 실행 전 별도로 다운로드해야 합니다.)

또한 게임 내 연출을 위해 GIF 파일을 사용하며,

이를 처리하기 위해 Pillow(PIL) 라이브러리가 필요합니다.

🔧 전선 퍼즐과 ROOM3 연결에 대한 설명
 
 본 게임에서는 방 2에서 진행되는 전선 퍼즐을 통해 발전기를 작동시키고,

이를 계기로 다음 공간으로 이동하는 구조를 의도하였다.

그러나 개발 과정에서 전선 클릭 판정과 자르기 상태를 안정적으로 처리하는 코드 구현에 어려움이 있어,

전선 퍼즐의 핵심 동작인 ‘전선을 자르는 기능’은 완전히 구현하지 못하고 건너뛰기버튼으로 대체하였다.

또한 맴 이동에서 room3으로 이동하지 못하는 오류가 발생하였다.

이는 프로젝트의 시간적 제약과 기능 구현에 집중하는 과정에서

스토리 연출 부분까지 충분히 확장하지 못한 결과이며,

향후 개선한다면 퍼즐 성공 후 컷신, 대사, 혹은 환경 변화 등을 통해

전선 퍼즐과 ROOM3 사이의 흐름을 더욱 자연스럽게 이어갈 수 있을 것이다.
